Funciones

Una función es una sección de código que calcula un valor o realiza una acción de manera independiente al resto del programa y que se puede repetir las veces que se desee dentro del código siempre y cuando sea necesaria. Esta se compone de tres elementos, primero los parámetros de entrada que son valores que recibe la función para posteriormente operar con ellos. Como segundo elemento, se tiene el código que se ejecuta dentro de la función, en este código se realizan las operaciones requeridas para cumplir con el objetivo de la función. Por último, se tiene el resultado o valor de retorno que es lo que se obtiene luego de ejecutar el código que hay dentro de la función.

Sintaxis

Siempre se debe de tener en cuenta que dependiendo del lenguaje en que se esté trabajando, las funciones se definirán de manera diferente. Para empezar se define el tipo de función que sea desea usar y se asigna un nombre a la función, este normalmente lleva un nombre que hace referencia al objetivo que cumple dentro del código. Luego, se declaran los parámetros de entrada dentro de un paréntesis, cabe aclarar que a cada parámetro se le define el tipo de dato. Por último, se implementa el código dentro de la función y se retorna el resultado que se obtiene de esta. Solo basta con llamar a la función dentro del código o función principal, ingresando los parámetros que requiere la función y de esta manera se obtendar el resultado deseado.
