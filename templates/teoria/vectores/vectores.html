Vectores

Los vectores en programación, al igual que en matemáticas y álgebra lineal, son un arreglo o una matriz de una dimensión que permite almacenar datos del mismo tipo de manera ordenada, además, este asigna una posición a cada uno de los datos llamada indexador. Cabe aclarar que esta posición o índice siempre es un entero positivo. Dependiendo del lenguaje, dichos arreglos empiezan desde index cero o index uno, por lo que se debe tener esto presente a la hora de programar.

Sintaxis

Aunque para cada lenguaje de programación se tiene una forma diferente de definir un vector, por lo general, lo que se hace es declarar el tipo de dato que contendra el arreglo, el nombre del arreglo y el tamaño de este el cual se pone entre corchetes. Por otra parte, se debe ir llenando posición a posición hasta completar el tamaño del arreglo, esto se puede hacer ya sea de forma manual o, de tal forma que se pueda llenar a medida en que se ejecuta el programa.
