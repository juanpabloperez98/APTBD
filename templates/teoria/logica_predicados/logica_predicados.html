Lógica de predicados

La lógica de predicados o lógica de programación es la base de todos los algoritmos, esto se debe a que todo programa en su forma más básica siempre se basa en una función lógica. Por más que se planteen fórmulas matemáticas, al final, más halla de la fórmula, se encuentra implicita una parte de la lógica. Hace parte de una decisión, por ejemplo si un valor es mayor que o menor que o igual que, y a partir de allí dar una información valiosa al usuario del software. Un ejemplo claro de esto, es la manera en la que se controla el flujo de ejecución de un programa a través de estructuras de selección o condicionales los cuales utilizan la lógica para saber si una condición se cumple o no y, de esta manera tomar un camino.

Sintaxis

Dependiendo del lenguaje de programación que se vaya a utilizar, dichas estrucutas de selección se definen o declaran de manera distinta. Pero básicamente, lo que se hace es poner la sentencia if con la condición que se quiere comparar dentro de corchetes. Posteriormente, se pone el código que se quiere ejecutar dependiendo de si la condición es verdadera o falsa.
