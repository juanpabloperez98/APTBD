Ejemplo #6


Cuando intentaba graficar y = |-x|, Benny razonó de esta manera:

El valor absoluto sólo hace que el valor sea positivo o cero.
Entonces, el resultado de |-x| es la versión positiva de x, o 0 si x = 0.
Pero el resultado de |x| es también la versión positiva de x, o 0 si x = 0.
Eso significa que |-x| = |x|.
Entonces la gráfica de y = |-x| es la misma que la gráfica de y = |x|.

¿Esta Benny usando el razonamiento inductivo?

A) No
B) Sí


Solución

Solución:


Benny no usó el razonamiento inductivo. Cuando usas el razonamiento inductivo, tomas ejemplos específicos y los generalizas a un conjunto más amplio de ejemplos. En este caso, las observaciones de Benny ya son generales, Si estuviera usando el razonamiento inductivo, podría notar ejemplos específicos como:

|-2| = 2, |-0.25| = 0.25, y |-(-3)| = |3|

A partir de eso podría generalizar que el valor absoluto del opuesto de un valor es el mismo valor o su opuesto, y entonces él podría hacer la conjetura |-x| = |x|. Nota que un caso general — la variable x — no es usada sino hasta el final del proceso, siendo que Benny usó la variable durante su razonamiento.
