Ejemplo #1

Dos números suman 25 y el doble de uno de ellos es 14. ¿Qué números son?

Solución

Solución:


x= primer número
y= segundo número
Los números suman 25:

    x + y = 25

El doble de uno de los números es 14:

    2x = 14

Se tiene el siguiente sistema:

     x + y = 25
    2x = 14

Se aplica sustitución:

    2x = 14  ->  x = (14/2) = 7
    x + y = 25  ->  7 + y = 25  ->  y = 25 - 7 = 18

Por lo tanto:

    x = 7
    y = 18
